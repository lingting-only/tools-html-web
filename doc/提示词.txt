提示词5：
谷歌AI Gemini3pro直接可用创作平台wendatu.cn 
角色设定：你是一位精通 Three.js、WebGL 视觉特效和计算机视觉（MediaPipe）的创意前端开发专家。
任务目标：编写一个单文件 HTML 应用程序，实现一个致敬《百变小樱》的 3D 魔法卡牌互动体验。
技术栈要求：
HTML/CSS：现代布局，深 ** 法风格（深紫/粉/金色调），包含加载动画、操作UI面板（上传图片、全屏、音乐控制）和右下角的摄像头预览框
Three.js (CDN)：用于核心 3D 渲染。
GSAP (CDN)：用于所有卡牌的平滑过渡动画。
MediaPipe Hands (CDN)：用于实时手势识别。
核心功能细节：
程序化纹理（无外部图片依赖）：
请使用 Canvas API 动态绘制复杂的魔法阵（包含外圈符文、内圈星形/月亮图案，带有发光效果），并将其作为纹理贴图应用到地面的 Plane 上，使其实时旋转。
使用 Canvas 绘制樱花花瓣纹理，并创建一个包含数百个粒子的粒子系统，模拟樱花飘落。
卡牌系统：
创建一叠 3D 卡牌（BoxGeometry）。
UI 面板允许用户<input type="file">上传多张图片作为卡面，上传一张图片作为卡背。
默认状态下卡牌是堆叠在一起的。
手势交互逻辑 (MediaPipe)：
张开手掌 (Palm)：触发“解除封印”，卡牌呈圆形（环绕摄像机）散开悬浮，并随手势左右挥动而旋转。
握拳 (Fist)：触发“收回卡牌”，卡牌迅速飞回并堆叠整齐。
剑指 (双指并拢)：当射线检测（Raycaster）指向某张卡牌时，该卡牌飞到屏幕正前方放大展示（查看模式）。再次张手则取消查看。
视觉特效
查看卡牌结束时，实现一个粒子消散效果（Dissolve effect），让卡牌化为光点消失或归位。
场景中添加辉光（Bloom）或点光源，营造梦幻氛围。
完善体验：
添加全屏切换按钮。
添加背景音乐播放控制。
在屏幕上显示当前检测到的手势提示（如“检测到：张手”）。
代码要求：
代码必须完整，包含所有 HTML、CSS 和 JS，无需额外配置即可在浏览器直接运行。
确保处理好 Canvas 绘制魔法阵时的边缘柔化，避免硬锯齿。
手势识别需要有一定的防抖动处理。
